## 自动化任务生成器 - 可视化桌面自动化工具

**项目地址：** https://github.com/QQHoney/zidong

> ⭐ **如果觉得有用，希望给个 Star 支持一下！** ⭐

### 这是什么？

一个**可视化**的桌面自动化工具，通过图形界面配置任务流程，自动生成 Python 脚本，无需编程基础。

### 核心功能

- **图片识别点击** - 截图目标按钮，自动识别并点击
- **OCR 文字识别** - 识别屏幕区域文字，支持变量传递
- **键盘鼠标模拟** - 输入文本、按键、组合键、长按（@hxy大佬公益站的签到）
- **定时任务** - 集成 Windows 计划任务，每日定时执行
- **代码导出** - 一键导出独立 Python 脚本

### 适用场景

- 网站自动签到/打卡
- 重复性表单填写
- 批量操作自动化
- 游戏/App 辅助脚本

### 快速开始

```bash
# 安装依赖
pip install customtkinter keyboard pyautogui pyperclip pillow numpy opencv-python requests

# 启动
python auto_task_gui.py
```

### 界面预览

左侧选择步骤类型 → 中间配置参数 → 底部实时预览代码 → 点击运行
